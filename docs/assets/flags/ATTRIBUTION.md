# Flag icons attribution

UI flag images in `app/src/main/res/drawable-nodpi/ic_flag_*.png` are rasterized from SVG sources in `app/src/main/assets/flags/`.

Source: [lipis/flag-icons](https://github.com/lipis/flag-icons) (MIT License).

| Locale tag | Language        | Flag file | ISO country |
|------------|-----------------|-----------|-------------|
| pt-BR      | Portuguese (BR) | br.svg    | BR          |
| en         | English         | us.svg    | US          |
| es         | Spanish         | es.svg    | ES          |
| fr         | French          | fr.svg    | FR          |
| zh-CN      | Chinese         | cn.svg    | CN          |
| ru         | Russian         | ru.svg    | RU          |
| de         | German          | de.svg    | DE          |
| ja         | Japanese        | jp.svg    | JP          |

Grid order in Settings matches the table above (Brazil first, then United States, Spain, France, China, Russia, Germany, Japan).
