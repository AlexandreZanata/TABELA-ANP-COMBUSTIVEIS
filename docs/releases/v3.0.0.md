### ANP Fuel Prices v3.0.0

Major update: register vehicles, see tank fill cost on Home, optional GPS city detection, station navigation, and local price drop alerts.

#### Highlights

- **Vehicle profiles (UC-010)** — up to 3 vehicles; tank capacity, fuel type, cheapest or specific station price reference
- **Tank fill cost (UC-011)** — Home cards show estimated full-tank cost for the active survey week
- **Optional location (UC-012)** — one-shot GPS + Nominatim reverse geocode after onboarding sync; OSM attribution shown
- **Station navigation (UC-013)** — Navigate button opens Maps/Waze with normalized station address
- **Price drop alerts (UC-014)** — local notifications after weekly import when tracked price falls vs previous week
- **Database migration** — v2.0.0 user data preserved when upgrading to v3 schema (`vehicle` table)

#### Install (sideload)

1. Download **`app-release.apk`** from the assets below.
2. **Upgrading from v2.0.0:** install over existing app — local price data migrates automatically.
3. Minimum Android **8.0** (API 26). Android **13+** may prompt for notification permission when enabling price alerts.

Build instructions: [docs/release-build.md](https://github.com/AlexandreZanata/TABELA-ANP-COMBUSTIVEIS/blob/main/docs/release-build.md)

#### Data & disclaimers

> **Fuel price data © [ANP](https://www.gov.br/anp)** — public government data.  
> **This app is not affiliated with ANP.**  
> Geocoding © [OpenStreetMap](https://www.openstreetmap.org/copyright) contributors (Nominatim).  
> Prices are weekly surveys, not live pump prices.
