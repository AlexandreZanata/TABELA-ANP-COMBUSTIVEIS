### ANP Fuel Prices v3.1.0

UI polish and contributor features: swipe between vehicles on Home, share the app, use GPS in the location picker, and back navigation on all secondary screens.

#### Highlights

- **Vehicle carousel on Home** — horizontal swipe between registered vehicles; tank cost card updates with selection
- **Use current location** — resolve municipality from device GPS in the location picker
- **Share app** — Settings share sheet with direct APK download link
- **Back navigation** — consistent back button on History, Prices, Search, Stations, Settings, and Location picker
- **Stale price warning** — banner when the latest ANP survey week may be outdated
- **Fixes** — first-launch sync reliability, system locale detection, vehicle station picker layout, Home scroll preservation

#### Install (sideload)

1. Download **`app-release.apk`** from the assets below.
2. **Upgrading from v3.0.0:** install over existing app — local data is preserved.

#### Full changelog

See [CHANGELOG.md](../../CHANGELOG.md#310---2026-06-23) for the complete list of changes.
