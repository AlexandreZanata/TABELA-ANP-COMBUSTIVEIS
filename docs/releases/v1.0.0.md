### ANP Fuel Prices v1.0.0

First public release of the open-source Android app for browsing official Brazilian ANP weekly fuel price surveys — fully on-device, no account required.

#### Highlights

- **Onboarding & sync** — First-run explanation and automatic download of latest ANP spreadsheets (UC-001, UC-002)
- **City search** — Autocomplete municipality search with state disambiguation (UC-003, UC-004)
- **Fuel averages** — Weekly gasoline, ethanol, and diesel prices for any city (UC-005)
- **Price history** — Local trend chart for selected location (UC-006)
- **Station detail** — Optional per-station download sorted by price (UC-007)
- **Settings** — Language (`en` / `pt-BR`), sync preferences, storage management, cache controls (UC-008)
- **Offline-first** — Browse cached data without network after first sync (BR-004)
- **Background sync** — WorkManager updates on Wi‑Fi when configured (BR-014)

#### Install (sideload)

1. Download **`app-release.apk`** from the assets below.
2. On your Android device, allow installation from your browser or file manager (“Install unknown apps”).
3. Open the app and complete the first sync when online.
4. Minimum Android **8.0** (API 26).

Signed **AAB** (`app-release.aab`) is included for maintainers preparing a Play Store upload.

Build instructions: [docs/release-build.md](https://github.com/AlexandreZanata/TABELA-ANP-COMBUSTIVEIS/blob/main/docs/release-build.md)

#### Data & disclaimers

> **Fuel price data © [ANP](https://www.gov.br/anp)** (Agência Nacional do Petróleo, Gás Natural e Biocombustíveis) — public government data.  
> **This app is not affiliated with ANP.**  
> Prices reflect **weekly surveys**, not live pump prices. Data freshness depends on ANP publication schedule.  
> The app works **offline after the first sync**; an internet connection is required only to download new weekly tables.

#### Privacy & license

- **No personal data collected** — see [privacy policy](https://github.com/AlexandreZanata/TABELA-ANP-COMBUSTIVEIS/blob/main/docs/privacy-policy.md)
- **Open source** — [MIT License](https://github.com/AlexandreZanata/TABELA-ANP-COMBUSTIVEIS/blob/main/LICENSE)
- **Source code:** https://github.com/AlexandreZanata/TABELA-ANP-COMBUSTIVEIS

#### Known limitations (v1.0.0)

- Weekly ANP data only — not real-time station prices
- Station detail is downloaded on demand and subject to local retention settings (BR-013)
- Play Store listing pending; GitHub Releases is the primary distribution path for v1.0.0

## Publishing steps (Phase 10.7)

1. Ensure `./gradlew validateReleaseTag` passes (version, license, disclaimers, clean tree).
2. Run `./gradlew createReleaseTag` — builds signed APK/AAB and creates annotated tag `v1.0.0` locally.
3. Push tag and branch when ready: `git push origin main && git push origin v1.0.0`.
4. Create GitHub Release from tag `v1.0.0` with title and body from this draft.
5. Attach `app-release.apk` (and optionally `app-release.aab`) as release assets.
