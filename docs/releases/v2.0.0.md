### ANP Fuel Prices v2.0.0

Major update: choose which ANP survey week to consult, search any Brazilian municipality nationally, edge-to-edge UI polish, and fuel product icons.

#### Highlights

- **Week picker (UC-009)** — Select latest or historical survey week before sync; gov.br catalog parity with operational notes
- **Active week chip** — Change week globally from any main screen
- **National search** — IBGE municipality catalog with intelligent FTS ranking (UC-004, BR-016, BR-017)
- **Safe areas** — Edge-to-edge layout respects status and navigation bars on all screens
- **Fuel icons** — Vector icon + label per fuel product; TalkBack reads localized names
- **Database migration** — v1.0.0 user price data preserved when upgrading to v2 schema

#### Install (sideload)

1. Download **`app-release.apk`** from the assets below.
2. **Upgrading from v1.0.0:** install over existing app — local price data migrates automatically.
3. Minimum Android **8.0** (API 26).

Build instructions: [docs/release-build.md](https://github.com/AlexandreZanata/TABELA-ANP-COMBUSTIVEIS/blob/main/docs/release-build.md)
